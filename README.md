# Stock_data_Analysis
# 📊 Sales Data Analysis Dashboard (Power BI + Excel)

## 🚀 Project Overview

This project focuses on analyzing sales data to uncover meaningful insights and support data-driven decision-making. Using Microsoft Excel for initial data preparation and Power BI for visualization, an interactive dashboard was developed to monitor key performance indicators (KPIs), identify trends, and evaluate business performance.

The goal of this project is to simulate a real-world business scenario where stakeholders need a clear and concise view of sales performance across different time periods.

---

## 🎯 Business Problem

Organizations often deal with large volumes of sales data but struggle to extract actionable insights. This project aims to:

* Track sales performance over time
* Identify peak sales periods and seasonal trends
* Analyze revenue and volume metrics
* Provide an intuitive dashboard for decision-making

---

## 🧰 Tools & Technologies Used

* **Microsoft Excel** → Data cleaning and preprocessing
* **Power BI Desktop** → Data modeling and visualization

## 📁 Dataset Information

* Source: Dataset provided in a YouTube project
* Format: Excel (.xlsx)
* Data includes:

  * Date / Time
  * Sales / Revenue metrics
  * Volume / Quantity
  * Additional calculated fields

---

## ⚙️ Project Workflow

### 1. Data Cleaning (Excel)

* Removed duplicates and handled missing values
* Standardized column formats
* Prepared dataset for analysis

### 2. Data Loading (Power BI)

* Imported cleaned Excel dataset into Power BI
* Verified data types and relationships

### 3. Data Modeling

* Created calculated columns and measures
* Structured data for efficient analysis

### 4. Dashboard Creation

* Designed an interactive dashboard using:

  * KPI Cards
  * Line Charts (trend analysis)
  * Bar Charts (comparisons)
  * Filters & slicers (Month, Year, etc.)

---

## 📊 Key Insights

* 📈 Sales show noticeable variation across months, indicating seasonal trends
* 🔝 Certain periods demonstrate peak performance, useful for business planning
* 📉 Some intervals reflect declining trends, highlighting potential areas of concern
* 📊 Volume and revenue patterns provide insights into demand fluctuations
